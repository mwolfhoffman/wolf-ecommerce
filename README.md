# wolf-ecommerce
Created to buildy SSR e-commerce platform quickly. 

To run:

1. ```npm i```

2. Run MongoDB locally

3. ```node keystone```

Will need .env file with Cloudinary credentials. Option for mailgun (currently not used).


Major dependencies 
1. Node.js/express
2. Cloudinary
3. Pug (formerly Jade)
4. Lodash
5. Stipe.js 

Highly customizable. Fork and use as needed.
